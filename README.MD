﻿# qt5.15.17

Customized version of Qt qt5.15.17.

## Features
- Use [VC-LTL 5.x](https://github.com/Chuyu-Team/VC-LTL5) toolchain to make the
  binary size smaller.
- Use [YY-Thunks](https://github.com/Chuyu-Team/YY-Thunks) toolchain to achieve
  the Windows XP support.

## System Requirements

- Supported OS: Windows XP or later
- Supported Platforms: x86, x86-64,arm64.

## Documents
I just built some LGPLv3 modules. 


If you want to use other modules, you can directly build from the source code, except for the webengine module because it requires a few code to be modified.
